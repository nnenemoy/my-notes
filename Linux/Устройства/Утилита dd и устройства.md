# Утилита dd и устройства

Единственная функция утилиты dd заключается в чтении из входного файла или потока и записи в выходной файл или поток с выполнением некоторых преобразований кодирования по мере необходимости. Одна особенно полезная функция команды dd в отношении блочных устройств заключается в том, что можно обрабатывать фрагмент данных в середине файла, игнорируя то, что происходит до или после.

Команда dd копирует данные в блоках фиксированного размера. 

Пример с символьным устройством: **dd if=/dev/zero of=new_file bs=1024 count=1**

Здесь один блок размером 1024 байта копируется из /dev/zero (непрерывный поток нулевых байтов) в файл new_file.

Важные параметры dd:
- if=file. Файл ввода. По умолчанию применяется стандартный ввод.
- of=file. Файл вывода. По умолчанию стандартный.
- bs=size. Размер блока. Команда dd считывает и записывает кол-во байтов данных за один раз. Чтобы сокращенно указать большие куски данных, можно с помощью b и k обозначить 512 и 1024 байта соответственно. Так в примере можно было указать bs=1k вместо bs=1024.
- ibs=size, obs=size. Размеры блока ввода и вывода. Если можно использовать один и тот же размер блока как для ввода , так и для вывода нужно применять bs, если нет - параметры ibs и obs.
- count=num. Общее кол-во блоков для копирования. Это нужно для больших файлов или бесконечных потоков данных, как /dev/zero, чтоб команда остановилась в какой-то точке. 
- skip=num. Параметр позволяет пропустить первые несколько (num) блоков в файле ввода или потоке и не копировать их в выходные данные.
