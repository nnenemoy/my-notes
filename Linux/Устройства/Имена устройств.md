# Имена устройств

Иногда бывает трудно найти имя устройства (например, при разбиении диска на разделы).

Способы узнать его:
- Выполнить udevd с помощью команды udevadm
- Найти устройство в каталоге /sys
- Угадать имя из выходных данных команды journalcrl -k (выводит сообщения ядра) или системного журнала ядра. Этот вывод может содержать описание устройств в системе.
- Для дискового устройства, которое уже видно системе, можно проверить вывод команды mount
- Командой cat /proc/devices можно посмотреть блочные и символьные устройства, для которых в системе в настоящее время имеются драйверы. Каждая строка состоит из номера и имени. Номер является основным номером устройства. Если можно угадать устройство по имени, нужно найти в каталоге /dev символьные и блочные устройства с соответствующим основным номером - и найдутся файлы устройств.

Среди этих способов надежен только первый, но он требует утилиты udev. При других способах, нужно иметь ввиду, что в ядре может не быть файла устройства для оборудования.
